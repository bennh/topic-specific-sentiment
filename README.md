# Discovering Topic-Specific Sentiment in Reuters Financial News

## Overview

This repository contains the code, data processing pipeline, and experimental results for the project:

**Discovering Topic-Specific Sentiment in Reuters Financial News**

The project extends traditional aggregate news sentiment analysis by decomposing financial news into latent topics and estimating sentiment at the topic level. Using Reuters financial news headlines, Latent Dirichlet Allocation (LDA), and FinBERT, we investigate how sentiment varies across economic and financial topics over time.

Rather than treating all news articles as a single sentiment signal, this project identifies topic-specific narratives and analyzes their sentiment dynamics, providing a richer understanding of financial news sentiment.

---

## Research Question

Most existing news sentiment indices aggregate sentiment across all news articles.

However, financial news covers diverse topics such as:

- Macroeconomy and Monetary Policy
- Banking and Financial Services
- US–China Trade Relations
- Big Tech and Digital Platforms
- Mergers and Acquisitions
- Energy and Commodity Markets

Aggregating sentiment across heterogeneous topics may mask important differences.

This project investigates:

> How does sentiment vary across different financial news topics, and what additional insights can be obtained from topic-specific sentiment analysis?

## Methodology

### Step 1: Text Preprocessing

Financial news headlines are cleaned using standard NLP preprocessing techniques:

- Lowercasing
- Punctuation removal
- Stopword removal
- Lemmatization

Implemented in:

```text
src/preprocessing.py
```

### Step 2: Topic Modeling

Latent Dirichlet Allocation (LDA) is applied to identify latent topics within Reuters financial news.

Main settings:

- Number of topics: 12
- Unigrams and bigrams
- Custom financial stopword list

Implemented in:

```text
src/topic_modeling.py
```

### Step 3: Sentiment Analysis

Sentiment is estimated using FinBERT.

**Reference**

Araci (2019), *FinBERT: Financial Sentiment Analysis with Pre-trained Language Models*

Sentiment categories:

- Positive
- Neutral
- Negative

For each article, FinBERT generates:

- Sentiment label
- Sentiment score
- Positive probability
- Negative probability
- Neutral probability

Implemented in:

```text
src/sentiment_analysis.py
```

### Step 4: Topic-Specific Sentiment Analysis

For each article:

1. Assign dominant topic using LDA
2. Predict sentiment using FinBERT
3. Aggregate sentiment within topics
4. Analyze temporal dynamics

Implemented in:

```text
src/visualization.py
```

## Key Findings

- Financial news sentiment differs substantially across topics.
- Financial Regulation and Corporate Scandals exhibit the most negative sentiment.
- COVID-19 and Manufacturing news is persistently negative during the sample period.
- Mergers and Acquisitions display the most positive sentiment.
- Topic-specific sentiment reveals information hidden by aggregate sentiment measures.
- FinBERT provides highly confident predictions for several domain-specific topics.
